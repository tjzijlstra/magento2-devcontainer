# n98-magerun2

[n98-magerun2](https://github.com/netz98/n98-magerun2) is a command-line toolkit for Magento 2 that provides a collection of useful utilities beyond what `bin/magento` offers. It is pre-installed in the PHP container.

## Usage

```bash
n98-magerun2 list
```

## Common Commands

| Command | Description |
| --- | --- |
| `n98-magerun2 db:console` | Open an interactive database shell |
| `n98-magerun2 db:dump` | Create a database dump |
| `n98-magerun2 db:import` | Import a database dump |
| `n98-magerun2 cache:flush` | Flush all caches |
| `n98-magerun2 admin:user:list` | List admin users |
| `n98-magerun2 sys:info` | Display system information |

For a full list of commands, see the [n98-magerun2 documentation](https://github.com/netz98/n98-magerun2#usage).
